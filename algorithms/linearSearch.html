Linear search

The linear search is the simplest search algorithm to implement. However it's not the most efficient. The search process consists on the sequential comparison of each element of the array, with the key element that needs to be found until, this last, is found.

The inefficency of this search algorithm increases with the increasing of the size of the array. If the key element is in the last position, in fact, the search will involve all the elements and, for big arrays, the process will become tedious.
